# chrome-extension-vue3-sample

## 概要

- Vue3で作成したChrome拡張機能のサンプルです。


## 動作環境

- Node.js - 16.x
- Yarn - 1.22.x

## ライブラリ

- vue - 3.2.x
- vue-router - 4.0.x
- @types/chrome - 0.0.190

## 動作確認

```bash
# パッケージインストール
yarn

# ビルド
yarn build
```
